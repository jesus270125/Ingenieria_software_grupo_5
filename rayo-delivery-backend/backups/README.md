# Directorio de Backups

Este directorio almacena los backups de la base de datos.

## Política de Retención
- Los backups se conservan durante 30 días
- Los backups más antiguos se eliminan automáticamente

## Tipos de Backup
- **Automáticos**: Se ejecutan diariamente a las 2:00 AM
- **Manuales**: Creados por el administrador desde el panel web

## Nomenclatura
Los archivos de backup siguen el formato: `backup_YYYYMMDD_HHMMSS.sql`
